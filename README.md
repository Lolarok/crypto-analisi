# 💰 Crypto Analisi Italia - WordPress Theme

Portale italiano crypto con notizie, analisi, segnali di trading.

## Design
- Bitcoin Orange #F7931A + Cyan #00E5FF su nero profondo
- Font: IBM Plex Mono + Space Grotesk
- Features: Price ticker live, signal cards, crypto watchlist

## Sezioni
- Notizie crypto in italiano
- Analisi di mercato
- Watchlist con segnali compositi
- Guida agli strumenti DeFi
- Portafolio suggerito

## Fonti RSS Crypto
- CoinDesk, CoinTelegraph, Decrypt
- The Defiant, The Block, Bankless
- CriptovaluteIO (italiano)
